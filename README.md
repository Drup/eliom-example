Eliom app example
=================

Setting up
----------

The only thing you will need is a recent [OPAM](http://opam.ocaml.org) (>= 1.0) for your platform.

After you have OPAM:

    opam switch 4.01.0
    eval `opam config env`
    opam install eliom

Building
--------

    make

Using
-----

    make server

Then, navigate to [http://localhost:5000](http://localhost:5000).
